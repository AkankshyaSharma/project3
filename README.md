# “NLP Sentiment Analysis on Customer Reviews”
Analyzed customer review data from e-commerce platforms to identify sentiment patterns. Performed data preprocessing, feature engineering, and exploratory data analysis (EDA) using Python. Applied NLP techniques such as text normalization, tokenization, stopword removal, and TF-IDF vectorization for effective feature extraction.
Developed and evaluated machine learning models including Naive Bayes and Logistic Regression for sentiment classification. Assessed model performance using accuracy metrics and selected the best-performing model. Deployed the final model using Streamlit, enabling real-time sentiment prediction through an interactive web application.

Tools & Technologies : Python, Pandas, NumPy, Scikit-learn, NLP, TF-IDF, Logistic Regression, Naive Bayes, Matplotlib, Seaborn, Streamlit, EDA, Text Preprocessing
